A modern, responsive e-commerce platform specializing in the sale of indoor and outdoor plants and gardening accessories, built to offer a calm and intuitive shopping experience.

Project Overview:
Green Nest is a front-end showcase of an e-commerce platform designed for plant enthusiasts. It emphasizes a clean, organic aesthetic using a calming emerald green color palette and modern UI practices.

What problem does it solve? It provides a clean, user-friendly, and visually appealing digital storefront for a specialty retail business, solving the problem of presenting natural products with a premium, trustworthy look online.

Who is it for? It's designed for plant lovers, home decorators, and businesses looking for a straightforward, beautiful interface to browse and purchase greenery.

What makes it unique? Its uniqueness lies in the detailed attention to branded UI/UX, featuring custom loading states, professional navigation, and strong visual hierarchy using the Tailwind CSS utility system, ensuring the brand identity is consistent across all elements.

Features:

User Authentication (Firebase Auth): Protected routes for the user profile, requiring login via a secure system (Firebase implementation assumed for production).

Responsive Design: Fully responsive layout ensuring optimal viewing and shopping experience across all screen sizes (mobile, tablet, desktop).
